# 🍕 Pizza Flutter App

A visually appealing **Pizza Ordering App UI built using Flutter**, showcasing modern UI design, smooth animations, and an intuitive user experience.

This project demonstrates how to build a beautiful food ordering interface using Flutter with clean architecture and responsive layouts.

---

## 🚀 Features

- 🍕 Browse different pizza items
- 🧾 View detailed product information
- ➕ Add items to cart (UI-based / logic if implemented)
- 🎨 Modern and attractive UI design
- ⚡ Smooth animations and transitions
- 📱 Responsive layout for multiple screen sizes

---

## 🛠️ Tech Stack

- **Flutter** – UI toolkit for building cross-platform apps  
- **Dart** – Programming language  
- **Material Design** – UI components  
- **Custom Widgets** – Reusable UI components  

---

## 📂 Project Structure

```

lib/
┣ models/        # Data models
┣ screens/       # App screens (Home, Details, Cart)
┣ widgets/       # Reusable UI components
┣ utils/         # Constants & helpers
┗ main.dart      # Entry point

````

---

## ⚙️ Getting Started

### Prerequisites

- Flutter SDK installed  
- Android Studio / VS Code  
- Emulator or Physical Device  

---



### 🔧 Installation

Clone the repository
```bash
git clone https://github.com/shubham230523/PizzaFlutterApp.git
```
Navigate into the project
```bash
cd PizzaFlutterApp
```
Install dependencies
```bash
flutter pub get
```
Run the app
```bash
flutter run
```
